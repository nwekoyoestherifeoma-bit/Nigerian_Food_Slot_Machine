# Nigerian Food Slot Machine

## Overview
A fun interactive slot machine game built in Python
where Nigerian foods replace traditional slot symbols.
Built entirely in Google Colab as my first Python game.

---

## How To Play
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. Enter a bet between ₦100 and ₦500
4. Spin and match 3 Nigerian foods to win!
5. Win 5x your bet if all 3 symbols match!

---

## Game Features
- Starting balance of ₦50,000
- Bet validation, rejects invalid amounts
- Random food selection every spin
- Win/loss balance tracking
- Play again option after each spin

---

## Nigerian Food 
- Jollof Rice
- Suya
- Moi Moi
- Egusi
- Peppersoup
